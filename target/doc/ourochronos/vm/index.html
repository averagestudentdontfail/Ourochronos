Virtual Machine for OUROCHRONOS epoch execution.

The VM executes a single epoch: given an anamnesis (read-only “future” memory), it produces a present (read-write “current” memory) and output.

The fixed-point search (in timeloop.rs) repeatedly runs epochs until Present = Anamnesis (temporal consistency achieved).
